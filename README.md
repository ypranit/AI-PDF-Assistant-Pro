## 🔥 Why I Built This

While learning AI Engineering, I realized that most AI tutorials stop at "Hello World."

I wanted to build something real.

This project started as a simple PDF chatbot and gradually evolved into a complete AI-powered document assistant that can analyze, summarize, compare, and interact with documents.

Along the way, I learned:

🧠 RAG Architecture

🔍 Semantic Search

⚡ FAISS Vector Databases

🤖 LLM Integration

🌳 Git & GitHub Workflows

🚀 Deployment with Streamlit Cloud
